# 🔍 Lost & Found Portal

A full-stack web application built with the **MERN Stack** that helps 
users report, browse, and manage lost and found items easily.

## ✨ Features

- 🔐 User Authentication & Protected Routes
- 📋 Post and manage lost/found items
- 🔎 Browse and search items
- 👤 User profile & dashboard
- 📱 Fully Responsive UI
- 🔗 RESTful APIs for frontend-backend integration
- 🗄️ MongoDB for secure data storage

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React.js, HTML5, CSS3 |
| Backend | Node.js, Express.js |
| Database | MongoDB |
| Other | REST APIs, JWT Authentication, Git |

## 🚀 How to Run Locally
```bash
# Clone the repository
git clone https://github.com/vidhivaja77/lost-and-found-portal

# Run Frontend
cd frontend
npm install
npm start

# Run Backend
cd backend
npm install
npm start
```
